# i18n 国际化实现计划

## 项目概况
- React 16 应用（class components）
- 使用 Tailwind CSS
- 使用 react-select 组件
- 需要支持多语言切换

## 实现步骤

### 1. 安装依赖
- `i18next` - 核心库
- `react-i18next` - React 绑定
- `i18next-browser-languagedetector` - 浏览器语言自动检测

### 2. 创建 i18n 配置
- `src/i18n/index.js` - 初始化 i18next，配置自动检测浏览器语言
- `src/i18n/locales/` - 语言文件目录
  - `en.json` - 英语
  - `zh.json` - 中文
  - 可扩展其他语言

### 3. 创建语言切换组件
- 使用现有的 `react-select` 组件
- 在 Header 组件中添加语言选择下拉框

### 4. 包装 App
- 在 `App.js` 中引入 i18n provider

### 5. 提取并替换文本
需要国际化的组件：
- `Header.js` - Logo、GitHub 链接、Twitter 链接
- `Home.js` - 主页所有文本
- `Editor.js` - 编辑器所有表单标签和按钮
- `Faq.js` - FAQ 页面

## 语言支持
- 默认：英语 (en)
- 自动检测浏览器语言
- 手动切换：英语、中文

## 实现顺序
1. 安装 i18n 依赖
2. 创建 i18n 配置和语言文件
3. 在 App.js 中集成 i18n
4. 修改 Header 添加语言切换器
5. 逐步替换各组件中的文本
